Scales Exercise

Exercise Instructions

In this exercise, you will create several plots using ggplot2 and modify axis labels and breaks using the scales library. Use the provided data set to complete the tasks.

Data Set

```{r}
# Create a data frame with proportions, revenue, and time variables
set.seed(123)
data <- data.frame(
  Category = rep(c("A", "B", "C", "D"), each = 5),
  Proportion = runif(20, 0.01, 1),
  Revenue = round(runif(20, 1000, 50000), 2),
  Time = rep(2015:2019, 4)
)

# Display the data
print(data)
```

Problem 1: Format Proportion Axis

Create a bar plot showing the proportion of each Category in the data for a specific year. Format the y-axis to display the proportions as percentages using the scales::percent function.

```{r}
ggplot(data, aes(x = Category, y = Proportion, fill = Category)) +
  geom_bar(stat = "identity") +
  scale_y_continuous(labels = scales::percent) +  # Modify labels to show percentages
  labs(title = "Proportions by Category", y = "Proportion", x = "Category") +
  theme_minimal()
```

Problem 2: Format Revenue Axis as Currency

Create a point plot showing Revenue over Time for each Category. Format the y-axis to display revenue values as dollars using the scales::dollar function.

```{r}
ggplot(data, aes(x = Time, y = Revenue, color = Category)) +
  geom_point(size = 3) +
  scale_y_continuous(labels = scales::dollar) +  # Format y-axis as dollars
  labs(title = "Revenue Over Time by Category", y = "Revenue ($)", x = "Year") +
  theme_minimal()
```

Problem 3: Custom Breaks for Time Axis

Create a line plot of Revenue over Time for each Category. Limit the x-axis to display only the years 2016, 2018, and 2019 using the breaks parameter.

```{r}
ggplot(data, aes(x = Time, y = Revenue, group = Category, color = Category)) +
  geom_line() +
  scale_x_continuous(breaks = c(2016, 2018, 2019)) +  # Custom breaks for x-axis
  labs(title = "Revenue Over Selected Years", y = "Revenue ($)", x = "Year") +
  theme_minimal()
```

Problem 4: Discrete Time Axis with Custom Labels

Create a point plot showing Proportion over Time for each Category. Customize the labels of the x-axis to display “Yr-2015”, “Yr-2016”, etc., using the labels parameter with a custom function.

```{r}
ggplot(data, aes(x = Time, y = Proportion, color = Category)) +
  geom_point(size = 3) +
  scale_x_continuous(labels = function(x) paste("Yr-", x, sep = "")) +  # Custom labels
  scale_y_continuous(labels = scales::percent) +  # Percent format for y-axis
  labs(title = "Proportion Over Time", y = "Proportion", x = "Year") +
  theme_minimal()
```

Problem 5: Log Transformation of Revenue

Create a bar plot of total Revenue for each Category across all years. Apply a log transformation to the y-axis and format it to display the transformed values in dollars using the scales::dollar function.

```{r}
ggplot(data, aes(x = Category, y = Revenue, fill = Category)) +
  geom_bar(stat = "identity") +
  scale_y_continuous(trans = "log10", labels = scales::dollar) +  # Log transformation + dollar format
  labs(title = "Log-Transformed Revenue by Category", y = "Log Revenue ($)", x = "Category") +
  theme_minimal()
```
